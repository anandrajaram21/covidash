Covidash

An open source, community driver, COVID-19 dashboard

Get Started

Easy to Use

Covidash was designed from the ground up to be easily used by anyone who wants to track the current progress of COVID-19.

Powered by Plotly

Covidash was built completely using Python, and Plotly Dash, which uses Flask under the hood, thus making the app highly performant.

Open Source

Anyone and everyone is free to submit a pull request and make contributions to the project. All contributions, small or large, are welcome.
